# Quick Start Guide

## How to Play Right Now

### Option 1: Direct Open (Simplest)
1. Download or clone this repository
2. Double-click `index.html`
3. Start playing!

### Option 2: Local Server (Recommended)
```bash
# Python
python3 -m http.server 8000

# Node.js
npx http-server

# Then open: http://localhost:8000
```

## Your First Playthrough

1. **Start Game** - Click the button on main menu
2. **Read Opening** - Press E to advance dialogue
3. **Move & Explore** - Use arrow keys or WASD
4. **Combat** - Approach enemies and attack with Z/J
5. **Make Choices** - When presented, use arrows + E
6. **Find NPCs** - Green characters have dialogue
7. **Find Portals** - Golden glowing objects change areas
8. **Defeat Bosses** - Purple enemies with crowns
9. **Reach Ending** - See your personalized conclusion

## Quick Tips

- **Double Jump**: Press Space twice to reach high platforms
- **Wall Climb**: Touch orange walls and use Up/Down
- **Dodge**: Shift gives temporary invincibility
- **Combo**: Attack rapidly for bonus damage
- **Save HP**: Dodge enemy attacks to preserve health
- **Explore**: Talk to all NPCs for story events
- **Choose Wisely**: Every decision affects your ending

## Troubleshooting

**Game won't load?**
- Use modern browser (Chrome/Firefox/Safari/Edge)
- Check browser console (F12) for errors
- Try local server instead of file://

**Controls not working?**
- Click on game canvas first
- Check keyboard layout
- Try alternative keys (Z/J for attack, etc.)

**Can't see full screen?**
- Zoom out (Ctrl/Cmd + -)
- Use larger monitor
- Game designed for 1280x720+

**Enemies too hard?**
- Use dodge more (Shift)
- Practice combo timing
- Use special attacks (X/K)
- Try tactical approach
